Sentinel II Hardened: 2D Time-Domain Voice Scrambler with Per-Frame Dynamic Seed Permutation for audio files

Sentinel II Hardened is a robust and efficient offline voice obfuscation tool that employs advanced segment permutation techniques combined with cryptographic key handling to enhance security. Designed for secure, air-gapped environments, this second-generation scrambler provides frame-specific, synchronized permutations that significantly improve resistance against casual and AI-assisted analysis.

Requirements:
Python 3.6+
numpy (for array operations and RNG)
scipy (for signal processing and I/O)



Usage Example:

python sentinel2hardened.py --action scramble --input clear.wav --output scrambled.wav --key AABBCCDDEEFF00112233445566778899
python sentinel2hardened.py --action descramble --input scrambled.wav --output recovered.wav --key AABBCCDDEEFF00112233445566778899

--action        scramble | descramble
--input         sample.wav
--output        scrambled.wav
--key           128 bits, 32 hexadecimal characters
--block_size    default= 15 (row/column scrambling matrix). For increased security use 20-40.
